Wired Acties

Hoe werken wired acties?

WIRED Actie: Persoon stapt op Meubi

Actie wordt gestart als een persoon op een Meubi gaat staan of zitten.

WIRED Actie: Persoon stapt van Meubi

Actie wordt gestart als een persoon van een Meubi afstapt.

WIRED Trigger: Bot bereikt user

Effecten worden geactiveerd wanneer de geselecteerde Bot een user bereikt

WIRED Trigger: Bot bereikt Meubi

Effecten worden geactiveerd wanneer de geselecteerde Bot één van de geselecteerde Meubi bereikt

WIRED Actie: Persoon zegt het goede woord

Actie wordt gestart als een persoon een speciaal woord of speciale zin zegt.

WIRED Actie: Persoon dubbelklikt op Meubi

Actie wordt gestart als de staat van een Meubi is veranderd door erop te dubbelklikken.

WIRED Actie: Persoon komt kamer binnen

Actie wordt gestart als Habbo een kamer binnenkomt.

WIRED Actie: Herhaal reactie

Reactie wordt elke x seconden geactiveerd.

WIRED Actie: Op moment x

Actie wordt gestart op een vast moment nadat de timer is gereset.

WIRED ACTIE: Score Behaald

Wordt geactiveerd als de opgegeven score behaald is

WIRED Actie: Game stopt

Actie wordt gestart als een game met een Timer Meubi is afgelopen in de kamer.

WIRED Actie: Game begint

Actie wordt gestart als een game begint met een Timer Meubi.

WIRED ACTIE: Botsing

Wordt geactiveerd als een (met de Wired Volg of Vlucht) verplaatste Meubi op een Persoon botst; de activerende gebruiker is de Persoon die botst.

WIRED ACTIE: Regelmatig Lang

Langere games! Actie wordt gestart telkens als de opgegeven tijd om is. Zoals de Actie Regelmatig, maar dan langer!
